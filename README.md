## Installation
> Just copy and paste the bellow command in your Termux... 
```
pkg up && pkg install wget -y && wget https://raw.githubusercontent.com/Anon4You/webinfo/main/webinfo.sh && chmod +x webinfo.sh && ./webinfo.sh
```
